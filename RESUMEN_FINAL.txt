 🔄 FLUJO DE LOGIN
═══════════════════════════════════════════════════════════════════════════════

1. Usuario abre http://localhost
   ↓
2. Nginx sirve index.html (Angular)
   ↓
3. App Angular carga en navegador
   ↓
4. Usuario ingresa credenciales
   ├─ Email: admin@tienda.com
   └─ Password: admin123
   ↓
5. Angular → POST /api/auth/login
   ↓
6. Nginx proxea a http://tienda-api:8080/api/auth/login
   ↓
7. API .NET:
   ├─ SELECT usuario FROM auth_db WHERE email = 'admin@tienda.com'
   ├─ BCrypt.Verify(password, hash)
   ├─ GenerarJWTToken()
   └─ RETURN {token, rol, expira}
   ↓
8. Frontend Angular:
   ├─ localStorage.setItem('auth_token', token)
   ├─ localStorage.setItem('auth_rol', rol)
   ├─ AuthInterceptor activo
   └─ Navigate to /dashboard
   ↓
9. AuthInterceptor en peticiones posteriores:
   ├─ GET token de localStorage
   ├─ Agregar header: Authorization: Bearer {token}
   └─ Enviar petición con JWT
   ↓
10. API valida JWT
    ├─ Validar firma
    ├─ Validar expiración
    ├─ Validar issuer/audience
    └─ Procesar petición
    ↓
11. Respuesta con datos
